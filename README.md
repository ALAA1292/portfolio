
# 🚀 Professional Portfolio Website

A clean, responsive, and modern personal portfolio website built to showcase professional skills, services, and projects.  
The website focuses on a smooth user experience, clear navigation, and a visually engaging layout.

---

## 🌟 Features

- **Responsive Design**  
  Fully responsive layout optimized for desktops, tablets, and mobile devices using Bootstrap 5.

- **Hero Section**  
  A welcoming landing section with a professional introduction.

- **About Me**  
  Highlights professional background, skills, and experience.

- **Services Section**  
  Organized presentation of services such as Web Development and UI Design.

- **Portfolio Gallery**  
  Displays projects with images and categories.

- **Contact Section**  
  A ready-to-use contact area for client communication.

- **Interactive UI**  
  Smooth scrolling and dynamic elements powered by JavaScript.

---

You can easily customize the template:

- Replace images inside the `imgs/` folder
- Modify styles in `css/main.css`
